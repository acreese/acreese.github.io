# Portfolio site (Hugo)

This repo hosts the Hugo content for alex-reese.com. The current working branch is **work**.

## How to see the latest copy updates
1. Fetch changes and check out the `work` branch:
   ```bash
   git checkout work
   git pull
   ```
2. Start the site locally to confirm the new text:
   ```bash
   hugo server -D
   ```
   Then open http://localhost:1313 in a browser.
